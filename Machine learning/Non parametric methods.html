Non-parametric methods

  • In cases where a parametric method might not be valid (e.g. modelling a multimodal distribution with a unimodal model), we can use a non-parametric method. These make few assumptions about the distribution being modelled.

Histogram methods

  • Identify a number of bins with width delta in each dimension.
  • A lower delta is better, but the catch is the lower the delta, the more data required. So its a balancing act between these two things. If there's no data in a bin we'll see a zeroing effect, which ruins our model somewhat.
