Information theory and entropy

Decision trees

  • Decision tree learning is not guaranteed to result in a global minimum
  • With information gain as a heuristic, hill climbing search over all possible decision trees (greedy search - no backtracking)
  • Some biases: shorter trees are better, higher information gain attributes placed near the root
