Latent variable models/mixture models

Motivation

  • Classic directed graphical models can get extremely complicated, e.g. in a healthcare setting. With a large amount of variables and parameters, calculating each conditional probability and joint probabilities becomes extremely difficult.
  • To combat this issue we can introduce latent variables which act as bottlenecks, compressing the data into a series of singular points. Example:
